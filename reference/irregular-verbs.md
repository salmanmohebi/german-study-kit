# Unregelmäßige Verben — Top 80

The most frequent strong/mixed verbs in German. Memorize the **three principal parts** (Infinitiv → Präteritum → Partizip II) plus the **helper verb** (*haben* or *sein*).

This file is the source for the **`German::Verbs::Unregelmäßig`** Anki deck — run `make verbs` to rebuild it.

## How to read the table

| Spalte | Beispiel | Hinweis |
|---|---|---|
| Infinitiv | gehen | the dictionary form |
| Präteritum | ging | 3rd-person sg. (drop the *-e* if there is one) |
| Partizip II | gegangen | for Perfekt |
| Hilfsverb | sein | which auxiliary the verb uses in Perfekt / Plusquamperfekt |
| Bedeutung | to go | English meaning |

For verbs with multiple meanings (e.g. *fahren* both transitive *haben* and motion *sein*), the most common usage is listed.
