# Goethe-Zertifikat B2 — Format & Strategie

## Überblick

| Item | Wert |
|---|---|
| Module | 4 — Lesen, Hören, Schreiben, Sprechen |
| Punkte | 100 pro Modul |
| Bestehensgrenze | **60 % pro Modul** — Module einzeln bestanden |
| Modular ablegbar? | Ja |
| Schriftliche Prüfung | ~190 Min |
| Mündliche Prüfung | ~15 Min (paarweise) oder Einzelprüfung möglich |

Format überarbeitet 2019 — verifiziere stets auf [goethe.de](https://www.goethe.de).

## Lesen (65 Min, 100 Punkte)

**5 Aufgaben:**
1. **Blog / Reportage** — Multiple Choice zum Detail
2. **Zeitungsartikel** — Aussagen ja/nein/nicht im Text
3. **Mehrere kurze Texte** zu einem Thema — Aussagen den Texten zuordnen
4. **Kommentar / Meinungsartikel** — Multiple Choice zur Position des Autors
5. **Formelle Mitteilung** (z.B. Geschäftsbrief) — Lückentext / Schlüsselwortsuche

**Strategie**: Texte sind länger als auf B1. **Zuerst Fragen lesen**, dann gezielt im Text Antworten suchen. Zeit ist knapp.

## Hören (40 Min, 100 Punkte)

**4 Aufgaben:**
1. **5 kurze Beiträge** (Anrufbeantworter / Durchsagen) — Multiple Choice. **Nur einmal gehört**.
2. **Interview** — Multiple Choice. Zweimal gehört.
3. **Diskussion / Talkshow** mit mehreren Sprecher*innen — wer äußert welche Meinung?
4. **Vortrag / Vorlesung** — Stichpunkte / Notizen ergänzen

Aufgabe 1 ist die schwierigste — keine zweite Chance. Aufgabe 4 testet längeres Konzentrationsvermögen.

## Schreiben (75 Min, 100 Punkte)

**2 Aufgaben:**
- **Aufgabe 1** (50 Min, ~150 Wörter) — Forumsbeitrag / Stellungnahme zu einem Thema. Diskussion: pro/contra, eigene Meinung.
- **Aufgabe 2** (25 Min, ~100 Wörter) — formelle E-Mail / Beschwerdebrief / Reklamation

**Was bewertet wird**:
- Inhaltliche Vollständigkeit (alle Aufgabenpunkte!)
- Kommunikative Angemessenheit (Register: formell vs. informell)
- Wortschatz: Vielfalt + Präzision
- Grammatik: B2-Strukturen aktiv einsetzen — Konjunktiv, Passiv, Nebensätze, Konnektoren
- Aufbau: Einleitung — Hauptteil — Schluss
- Sprachliche Korrektheit

**Tipp**: für Aufgabe 1 — Argumentationsgerüst:
> *Heutzutage wird oft diskutiert, ob… / Einerseits… andererseits… / Aus meiner Sicht… / Zusammenfassend lässt sich sagen, dass…*

## Sprechen (~15 Min, paarweise, 100 Punkte)

**2 Teile:**
1. **Vortrag halten** zu einem Thema (~4 Min) — Einleitung, Vor-/Nachteile, eigene Meinung, Beispiele aus dem eigenen Land. Kandidat*innen wählen aus 2 Themen.
2. **Diskussion** mit dem*der Partner*in (~5 Min) — kontroverses Thema. Eigene Position vertreten, auf Argumente reagieren, Kompromiss finden.

**Strategie**:
- Strukturwörter für Vortrag: *zunächst, im Folgenden, abschließend, einerseits, andererseits, im Vergleich dazu, insgesamt*
- Diskussions-Wendungen: *Da bin ich anderer Meinung. — Das sehe ich genauso. — Aber bedenke doch, dass… — Verstehe, aber…*

## Was du auf Sprachebene können musst

- Wortschatz: ~3 500 aktive Wörter — abstrakte Themen, idiomatische Redewendungen
- Grammatik: alles aus `grammar/A1`-`grammar/B2/` — besonders Konjunktiv I/II, Passiv, erweiterte Partizipialkonstruktionen, Nebensätze aller Art, Funktionsverbgefüge
- Themen: Bildung, Arbeitswelt, Medien & Digitalisierung, Umwelt, Gesellschaft, Politik, Wissenschaft, Kunst, Migration, Familie

## Konkreter Vorbereitungsplan (4-6 Monate vor Prüfung)

| Wochen vor Prüfung | Was machen |
|---|---|
| **20-24** | Grammatik B2 abschließen (alle 18 Topics), B2-Wortliste anfangen |
| **16-20** | Schreiben: 1 Forumsbeitrag / Woche zu B2-Themen, korrekturlesen lassen |
| **12-16** | Lesen-Training: 1 längerer DW-Artikel pro Tag, Wortschatz mining |
| **8-12** | Erster Modellsatz — modulweise. Schwächen identifizieren. |
| **6-8** | Hören intensiv: Tagesschau, Auf Deutsch gesagt!, DW-Reportagen |
| **4-6** | Sprechen üben: monothematische Vorträge (3-4 Min) — italki-Tutor*in |
| **2-4** | Zweiter Modellsatz unter Prüfungsbedingungen |
| **1-2** | Sprechen-Generalprobe, schwächstes Modul polieren |
| **Letzte Woche** | Routine. Schlaf. Keine neuen Inhalte. |

## Offizielle Modellsätze

Suche auf [goethe.de](https://www.goethe.de) nach **"Goethe-Zertifikat B2 Modellsatz"** — kostenlose PDFs + Audiodateien + Lösungen + Bewertungskriterien. Plane **mindestens drei** Modellsätze ein, einen davon zwingend unter Prüfungsbedingungen.

## Häufige Fallen

- **Schreiben**: zu informeller Stil bei Aufgabe 2 (formelle Mail) → Punktabzug. *Sie* statt *du*, formelle Anrede + Schluss.
- **Sprechen Teil 1**: zu reines Auflisten von Vor-/Nachteilen ohne eigene Meinung → niedrigere Punktzahl
- **Sprechen Teil 2**: nur die eigene Position halten ohne auf Partner zu reagieren → kommunikativ unangemessen
- **Lesen**: Aufgabe 4 (Meinungsartikel) — Position des Autors vs. zitierte Gegenmeinungen unterscheiden
- **Hören Aufgabe 1**: nicht ins Stocken kommen — bei verpasster Antwort weiterhören, Lücke nachher schließen
- **Modulares Bestehen**: gleiche Logik wie B1 — schwächstes Modul priorisieren. Falls ein Modul knapp ausfällt, kannst du es einzeln nachschreiben.
