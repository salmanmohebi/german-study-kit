# German Learning Resources

A curated, opinionated guide. The internet has thousands of German learning resources — most are mediocre. This file lists the ones that consistently work, organized two ways:

1. **My top picks per level** — if you only had time for one resource per skill
2. **Exhaustive lists by skill** — to dig deeper or swap when bored

**How to use this list**: don't try everything. Pick **1-2 per skill** and stick with them for at least 2-3 months before adding anything new.

---

## Top picks per level

If I had to recommend one of each, no compromise:

### A1 → A2 (your current focus)

| Skill | Top pick | Why |
|---|---|---|
| Listening | **Coffee Break German** (podcast) | English-explained, structured A1-A2 progression |
| Reading | **Dino lernt Deutsch** series by André Klein | Graded readers — first novel-style reading |
| Speaking | **Tandem app** + occasional **italki** lessons (€8-15/hr) | Free + paid combo; speak from week 2 |
| Writing | **LangCorrect** | Free; write 3-5 sentences/day, native speakers correct |
| Grammar | This repo + **Lingoni German** (YouTube, Jenny) | A1-B2 structured videos, very clear |

### A2 → B1

| Skill | Top pick | Why |
|---|---|---|
| Listening | **Slow German mit Annik Rubens** + **Easy German Podcast** | Slow G for clarity, Easy G for natural speech |
| Reading | **Nachrichtenleicht** (Deutschlandfunk, free) | Weekly news in plain German with audio |
| Speaking | **italki** weekly (1 hr/week) + **Tandem** daily | Tutor for correction, Tandem for volume |
| Writing | **LangCorrect** + **DeepL Write** | LangCorrect for human feedback, DeepL for self-checking |
| Grammar | **Lingolia.com** (free reference) + this repo | Topic search for any grammar point |

### B1 → B2

| Skill | Top pick | Why |
|---|---|---|
| Listening | **Auf Deutsch gesagt!** (podcast) + **DW Langsam gesprochene Nachrichten** | Long-form interviews + daily slow news |
| Reading | **DW Top-Thema mit Vokabeln** + start native (Spiegel, Zeit) | DW for guided, Spiegel for stretch |
| Speaking | **italki** 2x/week with same tutor | Building fluency requires repetition with one person |
| Writing | **italki notebooks** + **LangCorrect** | Long-form writing with corrections |
| Grammar | This repo (B1) + **Hammer's German Grammar** as reference | Hammer's = most comprehensive English-language grammar |

### B2+

| Skill | Top pick | Why |
|---|---|---|
| Listening | **Lage der Nation** (politics, native), **Tagesschau in 100 Sekunden** | Native register, real topics |
| Reading | **Spiegel, Zeit, Süddeutsche** (native journalism) | Stop using graded materials |
| Speaking | **italki** + meet Germans IRL (Stammtisch, Tandem) | Tutor for polish, IRL for confidence |
| Writing | Goethe B2 mock essays, corrected by tutor | Specific to exam if targeting Goethe |
| Grammar | This repo (B2) + reference Hammer's when stuck | At this level, you read more than you study rules |

| Platform | Typical rate | Strength |
|---|---|---|
| **italki** | €5-25/hr | Huge tutor pool, free trial lessons |
| **Preply** | €8-30/hr | Similar to italki, slightly more structured |
| **Lingoda** | ~€10-15/hr (group) | Group classes, sprint program |

### Self-practice

- **Pimsleur German** — 30-min audio lessons with pause-to-respond drilling
- **AI conversation** — Claude / ChatGPT can do role-plays, ask you questions in German, correct your grammar (limited but free)
- **Shadowing** — pick a 2-min podcast clip, repeat each sentence after the speaker, copy intonation

## A realistic week — what should you actually do?

For your stated 20-30 min Anki + 20-30 min listening, here's what I'd build out over time:

### A1 → A2 phase

| Day | Time | Activity |
|---|---|---|
| Daily | 25 min | Anki review (this repo's grammar deck + your vocab deck) |
| Daily | 20 min | Coffee Break German episode (podcast, while commuting/cooking) |
| 2x/week | 30 min | italki tutor (€8-15/hr) — start month 2 |
| Weekly | 15 min | Write 3-5 sentences in LangCorrect, get corrected |
| Monthly | 1 hr | Goethe A2 mock test section (under timer) |

### A2 → B1 phase

| Day | Time | Activity |
|---|---|---|
| Daily | 30 min | Anki + this repo's themen deck |
| Daily | 25 min | Easy German Podcast / Slow German episode |
| 2x/week | 30 min | italki tutor — same one for continuity |
| 3x/week | 20 min | Read 1 Nachrichtenleicht article, mine 3 sentences for Anki |
| Weekly | 30 min | Write a 80-100 word email/forum post → LangCorrect |

### B1 → B2 phase

| Day | Time | Activity |
|---|---|---|
| Daily | 30 min | Anki review (mostly review, fewer new) |
| Daily | 30 min | Auf Deutsch gesagt! / DW news + take notes |
| 2x/week | 60 min | italki tutor — discussion-based |
| 3x/week | 30 min | Read Spiegel/Zeit article, write summary in German |
| Monthly | 2 hr | Goethe B1 or B2 mock essay + speaking practice |

## What NOT to do

- Don't subscribe to 5 apps. **Two max.**
- Don't watch grammar YouTube while working through this repo's grammar decks — it's redundant.
- Don't start native journalism at A2 — you'll burn out. Use graded materials until B1.
- Don't skip writing because it's "harder than reading". Write in some form **every single week**.
- Don't binge-study one weekend, then go silent for two. Daily 30 min beats weekly 4 hours.
- Don't pay for Babbel/Lingoda if you have this repo + free podcasts + a tutor. Money is better spent on tutoring.

## My one-line recommendation

**A1-A2**: Coffee Break German + this repo + Tandem + LangCorrect + a textbook (Schritte / Begegnungen) if you want structure.
**A2-B1**: Easy German Podcast + this repo + italki weekly + Nachrichtenleicht for reading.
**B1-B2**: Auf Deutsch gesagt! + this repo + italki 2x/week + Spiegel/Zeit + Goethe B2 prep.
